API

jnt-row

Property Description Type Default
[justify] - start | end | around | between | evenly start
[align] - start | end | stretch | baseline | center stretch
[direction] - row | column | row-reverse | column-reverse row
[alignContent] - start | around | between | center | end | evenly | stretch start
[wrap] - wrap | noWrap | reverse wrap

jnt-col

Property Description Type Default
[alignSelf] - center | stretch | start | baseline | end | auto auto
[span] - number 1
[mobile] < 768px number null
[tablet] < 992px number null
[desktop] < 1200px number null
[wide] <= 1200px number null
