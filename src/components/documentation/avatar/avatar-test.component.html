Avatar

Requirements

  1. The avatar must have different sizes: tiny, small, normal, large.
  2. The avatar must have different shapes: circle, square.
  3. Avatar can contains icon, text or image.
  4. If image passed it must replace icon or text.
  5. If image or icon are not passed avatar must use first char of text.
