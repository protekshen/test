Junte UI

Very strong framework!

Junte UI is a collection of quality UI components for Angular.

Angular Support

Angular ^7.1.0

Version

npm v0.0.117

Installation

Install Junte UI with npm :

Once installed you need to import our main module.

Import style and SVG icon assets file link in angular.json

Import the style files of the components you need into the global style file style.scss:

If you need all the components, then import the file components.scss:

Some components use angular animation, you need to import BrowserAnimationsModule to your application

Supported Browsers

Modern browsers and Internet Explorer 11+

Themes Support

Style files are stored separately from the components, so any style can be changed and the project can be built.

For development, we use Sass variables , which you can customize according to personal preferences.

Sources
